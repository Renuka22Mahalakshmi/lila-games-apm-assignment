# LILA BLACK: Player Journey Analytics Tool

An interactive, web-based visualization dashboard built for Level Designers to analyze player telemetry, movement patterns, and combat high-density zones in LILA BLACK.

This repository contains a curated sample of 40 match files across the Ambrose Valley and Lockdown maps to demonstrate the tool's visualization capabilities while keeping the repository lightweight. Full production support for Grand Rift is integrated into the architecture and will populate automatically upon connecting the full 1,200+ file dataset.

## 🚀 Live Demo
**Access the hosted tool here:** https://lila-games-apm-assignment-renuka.streamlit.app/

---

## 🛠️ Tech Stack
- **Language:** Python 3.12
- **Dashboard:** Streamlit
- **Data Visualization:** Plotly (Scatter & Density Heatmaps)
- **Data Processing:** Pandas, PyArrow (Parquet handling)
- **Image Processing:** Pillow (PIL)
